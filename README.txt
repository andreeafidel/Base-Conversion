	In eticheta base se compara daca ecx a depasit valoarea lui nums. Se muta in eax valoarea din nums_array si in ebx valoarea corespunzatoare bazei din base_array. Daca baza nu e valida sare la eticheta care afiseaza mesajul "Baza incorecta".
	In eticheta impartire, se imparte numarul stocat in eax la cel stocat in ebx. Restul stocat in edx se compara cu 9. Daca este mai mare sare la eticheta bigger_10 unde i se adauga 87 pentru a da litera corespunzatoare la afisarea caracterului cu PRINT_CHAR. Apoi se adauga edx pe stiva, iar daca eax nu e 0 continua impartirea numarului din eax. La fel, daca restul era mai mic decat 9, i se adauga codul ASCII al lui '0' pentru a afisa cifra sub forma de caracter. Apoi, se adauga pe stiva, se compara eax cu 0 si se continua impartirea. Cand eax a ajuns 0 se incrementeaza ecx si se sare la eticheta print.
	In eticheta print daca stiva s-a terminat printeaza o linie noua, altfel scoate de pe stiva resturile si le printeaza.
